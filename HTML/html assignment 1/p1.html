b tag

The b HTML element is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. This was formerly known as the Boldface element, and most browsers still draw the text in boldface. Basicaliy it is used to Bold any specific word or parageaph.

i tag

The i HTML element represents a range of text that is set off from the normal text for some reason, such as idiomatic text , technical terms, taxonomical designations, among others.

u tag

The u HTML element represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation. This is rendered by default as a simple solid underline .

select tag

The select HTML element represents a control that provides a menu of options.

button tag

The button HTML element is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology.
